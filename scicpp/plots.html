plots¶

Plotting functions defined in header <scicpp/plots.hpp>.

Base on the Sciplot library, which is a header only library. Gnuplot is required to run the executable. This is only required to use the plots module, not other scicpp modules.

Extends Sciplot with some specialized plots available in Matplotlib (hist, psd, cohere, csd)

Statistics¶

hist
Plot a histogram.
boxplot
Boxplots.

Spectrum¶

csd, psd
Plot the spectral density.
cohere
Plot the coherence between two signals.
